HTML vs HTML5

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 is the fifth version of HTML, introduced in 2014.
HTML was introduced in the early 1990s. HTML5 introduces new semantic elements like <header>, <footer>, <nav>, and <article> for better document structure.
HTML4 and its previous versions had limited support for multimedia elements. HTML5 has native support for audio and video elements, eliminating the need for third-party plugins like Flash.
